# 音频文件说明

请将以下音频文件添加到此目录：

## 需要的文件：

1. **move_piece.mp3** - 下子音效
   - 建议：清脆的木块碰撞声或棋子落盘声
   - 时长：0.5-1秒
   - 推荐来源：
     - Pixabay: https://pixabay.com/sound-effects/search/chess/
     - 免费音效网站

2. **capture_piece.mp3** - 吃子音效
   - 建议：稍重的棋子碰撞声
   - 时长：0.5-1秒
   - 可以与move_piece相同，或选择更有力的声音

3. **background_music.mp3** - 背景音乐
   - 建议：悠扬的中国传统音乐
   - 时长：2-5分钟（会循环播放）
   - 推荐乐器：古筝、二胡、笛子
   - 推荐来源：
     - Free Stock Music: https://www.free-stock-music.com/chinese.html
     - Fesliyan Studios: https://www.fesliyanstudios.com/royalty-free-music/downloads-c/chinese-music/61
     - Chosic: https://www.chosic.com/free-music/chinese/

## 音频格式要求：
- 格式：MP3（推荐）或WAV
- 采样率：44.1kHz或48kHz
- 比特率：128kbps以上

## 注意事项：
- 确保音频文件有适当的使用许可
- 文件名必须全小写，使用下划线分隔
- 放置在 app/src/main/res/raw/ 目录下
