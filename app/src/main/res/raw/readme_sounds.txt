音效文件说明 (Sound Files Guide)
================================

为了获得最佳的游戏体验，请在此目录添加以下音效文件：

需要的音效文件：
--------------
1. move.mp3 或 move.wav - 棋子移动音效
   建议：轻柔的木质敲击声，体现古典棋盘落子的感觉

2. capture.mp3 或 capture.wav - 吃子音效
   建议：略重的敲击声，表现力量感

3. check.mp3 或 check.wav - 将军音效
   建议：清脆响亮的声音，提醒玩家注意

4. gameover.mp3 或 gameover.wav - 游戏结束音效
   建议：优雅的结束音，可以是钟鼓声

5. select.mp3 或 select.wav - 选中棋子音效
   建议：轻微的点击声

音效获取建议：
-------------
1. 使用免费音效网站：
   - Freesound.org
   - ZapSplat.com
   - SoundBible.com

2. 搜索关键词：
   - "wood click"
   - "bamboo hit"
   - "stone tap"
   - "Chinese traditional"

3. 音效要求：
   - 格式：MP3 或 WAV
   - 长度：0.5-2秒
   - 音量：适中，不要太大
   - 风格：古典、优雅

注意：
-----
如果不添加音效文件，游戏将使用系统默认提示音作为替代。
添加音效文件后，需要重新编译应用才能生效。
