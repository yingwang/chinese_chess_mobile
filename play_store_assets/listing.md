## Full Description (4000 chars max)

中国象棋 — 一款精心打造的象棋应用，结合强大AI引擎与古典美学设计。

🎮 游戏模式
• 人机对战 — 5个AI难度等级，从初级到大师
• 双人对战 — 同设备本地对弈
• AI对战 — 观看AI自我博弈
• 残局练习 — 8个经典残局（重炮杀、铁门栓、天地炮、马后炮等）

🧠 专业AI引擎
采用现代博弈搜索技术：
• Alpha-Beta剪枝 + 空步裁剪
• 后期走法缩减（LMR）
• Zobrist哈希 + 置换表
• 杀手走法 + 历史启发式
• 静态搜索消除水平线效应
• 30+经典开局库
• 大师级搜索深度7层

📊 评估系统
• 子力价值 + 位置评估表
• 将帅安全评估
• 过河卒/卧槽马加分
• 车的活跃度评估
• 炮的残局调整
• 双马联动加分

🎨 古典界面
• AI生成写实木纹棋盘纹理
• 象牙质感3D凸起棋子
• 平滑走棋动画 + 触觉反馈
• 金色选中高亮
• 深色主题，统一古典风格

✨ 丰富功能
• 提示系统 — AI推荐最佳走法
• 自动存档 — 退出自动保存，下次继续
• 棋谱回放 — 逐步复盘
• 棋谱导出 — 分享给好友
• ELO评分 — 追踪棋力成长（初学→大师）
• 战绩统计 — 胜负平、胜率
• 古琴背景音乐
• 音效开关

🔒 隐私保护
• 无需网络连接
• 不收集任何个人数据
• 无广告、无追踪
